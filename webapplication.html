Karachi trading Co.

Created a application for Karachi Trading Co.
taking in consideration of the requiremnts i was able to develop a very well fit web application for Karachi Trading Co.

Front end and backend.

In this project, the front end is developed using React, which is a popular JavaScript library for building user interfaces. React is used to handle the dynamic state of the product list, search functionality, and the cart. The component structure allows for efficient management of product data, selection, and user interactions. The product details are displayed with the help of HTML and CSS, and dynamic updates are triggered by React's state management. The project uses localStorage to store the user's selected items for checkout. This application allows users to search for products, add them to the cart, view their selected quantities, and proceed to the checkout page. The list of products includes various tech and home appliance items, each with its name, price, and an associated icon from Font Awesome.

Login page.

This HTML code creates a simple login page that integrates React for dynamic behavior. It includes the React and ReactDOM libraries via CDN and uses Babel to transpile JSX directly in the browser. The page is styled with a full-screen background image that covers the entire viewport, while the login form is placed in a centered, semi-transparent box. The form elements such as input fields and buttons are designed with rounded corners and hover effects for a modern look. Additionally, the page features a fixed footer at the bottom with a license message, "Karachi Trading Co. since 2024." Overall, the code sets up a visually appealing and responsive login page using React and modern UI design principles.

purchase page.

This code is for a simple product listing and shopping cart functionality built with React. It allows users to browse through a list of products, search for specific items, and add them to their cart. The products displayed include various electronics and home appliances, each showing its name, price, and an icon. Users can increase the quantity of selected products and proceed to the checkout page, where the selected items are saved using localStorage.

Inventory page.

This HTML code is for a webpage designed to display an inventory management system using React. It integrates React and ReactDOM via CDN and utilizes Babel to transpile JSX directly in the browser, with the actual React code being loaded from an external file called Inventory.js. The page's layout is styled with a full-screen background image that covers the entire viewport. It uses a flexbox layout to vertically center the content.
