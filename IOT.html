ESP32 Cam security system.

This project sets up a door security system using an ESP32 with a camera module. It connects to Wi-Fi and the Blynk app for remote control. The system can take a photo when a button is pressed in the app or when a motion sensor (IR Sensor) is triggered. It also allows the door lock to be opened remotely via the app. The camera's feed is accessible through a local IP address, and photos are sent to the Blynk app. Additionally, the system activates a bell when motion is detected.

ESP 32 Cam

The ESP32 in the project acts as the central controller, handling Wi-Fi connectivity, communication with the Blynk app, and managing inputs from the motion sensor and camera module. It processes commands to capture images, trigger the door lock, and send notifications remotely.
