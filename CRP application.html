Computer Research Project.

making a application for CRP.
this application will take any word or centence entred and then it will encrypt it so that the data is save when its in transit or kept in a safe place.

Python.

The code performs encryption and decryption of a word using a simple cipher with a key "4387". When encrypting, it adds the corresponding key digit to the ASCII value of each letter. For decryption, it subtracts the key digit from the ASCII value. The result is either saved to a file or decrypted from a file based on user input.

Increpting a letter.

The code converts a letter to its ASCII value, adds 5 to it for encryption, and then converts it back to a letter. It also decrypts an encrypted letter by subtracting 5 from its ASCII value to retrieve the original letter.

Increpting a word.

The code encrypts a word using a repeating key by adding key digits to the ASCII values of each character, saves the encrypted word in a file, and then decrypts it by subtracting the key digits from the ASCII values to recover the original word.
